Frequently Asked Questions

How do I get a Google AI Studio API key?

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Get API Key" or "Create API Key"
  4. Copy the generated API key
  5. Paste it into the API key field on our homepage

Note: The API key is free to use with generous quotas. Your key is stored locally in your browser and never sent to our servers.

Is my data secure?

Yes! This application runs entirely in your browser. Your PDF files and API key never leave your device except for the direct API calls to Google's Gemini service using your own credentials.

We have no backend servers, so there's nowhere for your data to be stored or intercepted.

What file size limits are there?

The main limitation is your browser's memory and the Gemini API's token limits. Our automatic batching system splits large documents into chunks, so you can process books with hundreds of pages.

For very large files (100+ MB), processing may take longer and consume more API quota.

Does this work with scanned PDFs?

No, this tool requires PDFs with extractable text. Scanned PDFs (images of pages) need OCR processing first. You can use tools like Adobe Acrobat or online OCR services to convert scanned PDFs to text-based PDFs before using our converter.

How much does it cost?

The website itself is completely free. However, you'll need a Google AI Studio API key, which has:

  • Free tier with generous quotas (15 requests per minute for Gemini 1.5 Flash)
  • Pay-as-you-go pricing for usage beyond free tier
  • Very affordable rates (typically pennies per document)

Check Google AI pricing for current rates.

Which Gemini model should I use?

  • Gemini 1.5 Flash: Fastest and most cost-effective. Great for most documents.
  • Gemini 1.5 Pro: Higher quality output, better for complex technical content.
  • Gemini Pro: Balanced option for general use.

We recommend starting with Gemini 1.5 Flash and upgrading to Pro if you need better quality.

Can I customize the conversion process?

Yes! Visit the Settings page to:

  • Edit prompt templates
  • Adjust batch sizes
  • Modify temperature and other model parameters
  • Customize how code and figures are described
  • Set transformation rules

How long does conversion take?

Processing time depends on:

  • Document length (pages and word count)
  • Selected Gemini model (Flash is faster than Pro)
  • API rate limits (15 requests/minute for free tier)
  • Your internet connection speed

Typical times: 50-page book = 5-10 minutes, 200-page book = 20-40 minutes

What happens if processing fails?

If an error occurs, you'll see a detailed error message. Common issues:

  • Invalid API key: Check your key and try again
  • Rate limit exceeded: Wait a minute and retry
  • Network error: Check your internet connection
  • Quota exceeded: Check your API usage limits

The app will attempt to resume from where it failed when possible.

Can I use this offline?

Partially. Once you load the website, the interface works offline. However, you need an internet connection to make API calls to Google's Gemini service for the actual text transformation.

What browsers are supported?

Modern browsers with JavaScript enabled:

  • Chrome/Edge (recommended)
  • Firefox
  • Safari
  • Opera

Mobile browsers work but may struggle with very large files due to memory constraints.
